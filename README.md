# Developer Tools Bundle

一个强大的开发者工具包，包含批量重命名工具和代码质量检查工具，帮助开发者提升工作效率。

## 🚀 功能特点

### 批量重命名工具
- **序号重命名**：自动添加序号前缀
- **字符串替换**：批量替换文件名
- **日期重命名**：自动添加日期前缀
- **安全备份**：防止数据丢失
- **模式匹配**：支持通配符过滤

### 代码质量检查工具
- **多语言支持**：Python、JavaScript、Java等
- **自动检测**：识别代码质量问题
- **详细报告**：生成质量分析报告
- **快速扫描**：快速扫描整个项目
- **自定义规则**：支持自定义检查规则

## 📖 使用指南

### 快速开始

1. 下载并安装工具
2. 使用激活码解锁完整功能
3. 查看用户手册了解使用方法
4. 观看视频教程快速上手

### 批量重命名

```bash
# 序号重命名
python batch_renamer.py --mode index --pattern "*.jpg"

# 字符串替换
python batch_renamer.py --mode replace --old "old_" --new "new_"

# 日期重命名
python batch_renamer.py --mode date --format "%Y-%m-%d"
```

### 代码质量检查

```bash
# 检查整个项目
python code_quality_checker.py --project ./my_project

# 检查特定文件
python code_quality_checker.py --file ./main.py

# 生成报告
python code_quality_checker.py --output report.html
```
