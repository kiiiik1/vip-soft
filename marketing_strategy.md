# GitHub Marketplace 发布指南

## 产品信息
- **产品名称**: 开发者工具包
- **包含工具**: 批量重命名工具 & 代码质量检查工具
- **定价**: 个人版 $4.99, 专业版 $9.99, 企业版 $19.99
- **目标用户**: 开发者、项目经理、技术团队

## GitHub Marketplace 发布步骤

### 1. 准备 GitHub App
- 创建 GitHub App 应用
- 配置 OAuth 应用
- 设置 webhook 和权限

### 2. 应用开发
- 创建 GitHub Actions 工作流
- 实现安装和卸载钩子
- 开发 billing 和 pricing 功能

### 3. Marketplace 清单
- 创建详细的清单页面
- 包含截图、功能描述、定价信息
- 编写使用文档

### 4. 测试和验证
- 在测试环境中验证功能
- 确保支付流程正常
- 测试安装和配置过程

### 5. 正式发布
- 提交审核
- 设置发布计划
- 监控用户反馈

## Gumroad 发布准备

### 1. 产品页面
- 创建产品详情页
- 上传演示视频
- 设置价格和版本

### 2. 支付配置
- 集成支付宝支付
- 配置自动发货
- 设置退款政策

### 3. 营销材料
- 制作产品截图
- 编写使用教程
- 准备客户评价
