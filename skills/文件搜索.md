# 文件搜索技能

## Description
这个技能演示如何在文件系统中搜索和查找特定的文件。支持多种搜索模式和过滤条件。

## Usage
```bash
# 列出所有技能
python3 find_skills.py --list

# 搜索特定技能
python3 find_skills.py --search "文件"

# 查找技能文件
python3 find_skills.py --find "search"

# 获取技能详细信息
python3 find_skills.py --info "文件搜索"
```

## Features
- 支持按文件名搜索
- 支持按文件类型过滤
- 支持递归搜索
- 支持正则表达式匹配
